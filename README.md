# simul_pb
Simultaneous video playback app

### Requirements
* Python 3.14+
* [uv](https://github.com/astral-sh/uv) (Recommended) or pip
* libmpv-2.dll (download from https://mpv.io/ and place it on the folder where the simul_pb.py is located.)

### Installation & How to run
Clone the repository to an appropriate location.

#### Using uv (Recommended)
1. Run `uv sync --frozen` in the project directory to set up the environment.
2. Run `simul_pb_launcher.pyw`.

#### Using pip
1. Run `pip install -r requirements.txt` to install dependencies.
2. Run `simul_pb_launcher.pyw`.

### Supported format.
mp4, avi, mkv, mov, gif.

### Usage
Press F11 key to toggle fullscreen/windowed mode.

Press "Change Layout" button to switch movies layout cyclically as "1x4" -> "1x3" -> "1x2" -> "1x1" -> "2x2" -> "1x4" ...
